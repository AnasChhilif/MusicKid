# MusicKid
A next generation media player, with vim-like bindings
## Installation

* Clone the repo
    ```sh
    git clone <git-url>
    cd MusicKid/Final
   ```
* Install dependencies :
    ```sh
    sudo pacman -S sdl2 sdl2_image sdl2_mixer sdl2_ttf
	```
* Build MusicKid :
    ```sh
    make
    ```
## Features
* Playing music and displaying relevant info (Album cover, Track title, artist, etc...)
